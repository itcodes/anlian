    发行量10亿枚 （暂不上交易所，只用于社区贡献记录及融资，币上链Api与社区关联）

    运营： 负责社区整体运营，开发者管理维护; (12%/1.2亿枚)
    技术： 负责社区平台建设，区块链技术研发; (8%/0.8亿枚)
    投资： 社区赞助人或投资者; (10%/1亿枚)
    生态： 对社区贡献开发者; (30%/3亿枚) 
    预留： 用于预留额度; (20%/2亿枚)
    空投：用于新用户的激励； （10%/1亿枚）
